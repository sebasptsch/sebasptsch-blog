As any application gets larger and as the codebase grows along with it the way that state is managed becomes more essential to consider.

For frontend applications built using frameworks like React or Vue there are a number of different libraries that all try to accomplish the same goal, making sure that the way data changes across the application is predictable and able to be accessed where it's needed.
